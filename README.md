# Forest Fire Prediction - Algeria & Tunisia

This project predicts forest fire occurrence using soil and climate data from Algeria & Tunisia (2024).  
It includes from-scratch implementations of supervised learning models, evaluation metrics, feature selection, and parameter tuning using metaheuristics.

## Notebook
The main ML code is in `modelsH.ipynb` and 'Kmeans.ipnyb', which is fully executed with all graphs and results displayed.

## Features
- Data preprocessing and feature engineering
- Supervised learning models implemented from scratch:
  - K-Nearest Neighbors (KNN)
  - Decision Trees
  - Random Forest
- Feature selection and parameter tuning using metaheuristics
- Evaluation using accuracy, ROC-AUC, F1-score
- Comparison with scikit-learn implementations
- Optional unsupervised analysis (clustering)
